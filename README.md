# react-base-table-extends

## 在线 demo

https://52flutter.github.io/react-base-table-extends

## Getting Started

Install dependencies,

```bash
$ npm i react-base-table-extends
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
