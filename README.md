# React WebComponent Table

一个可以用于 Vue 的 React WebComponent Table 组件，适用于高性能的场景。

## 特性

- 简单易用，快速集成。
- 支持自定义配置。

## 安装

使用 npm 或 yarn 安装：

```bash
npm install @bdlite/table
# 或者
yarn add @bdlite/table
```

## 使用方法

1. 在项目中直接导入使用：

   ```javascript
   import { BaseTable } from '@bdlite/table';
   ```

2. 在 HTML 中使用：

   ```html
   <react-base-table></react-base-table>
   ```
